# MediConnect – Complete Healthcare Information Platform
MediConnect is a comprehensive multi-page healthcare awareness and assessment platform designed to provide reliable medical information, health assessments, 
emergency guidance, and interactive tools. All-in-one structured web application.

# Project Overview
MediConnect aims to:
    - Promote health awareness
    - Provide medical encyclopedic information
    - Offer interactive health calculators
    - Guide users in emergencies
    - Encourage preventive healthcare habits

# Technologies Used
HTML5 – Page structure
CSS3 – Styling & layout
JavaScript – Calculators, quiz logic, interactivity

# Key Functionalities
  # Interactive Health Quiz
    MCQ-based system
    Highlights correct & incorrect answers
    Displays total score
    Real-time validation
  # BMI Calculator
    Takes height & weight input
    Calculates BMI
    Displays health category
  # Heart Risk Calculator
    Risk evaluation logic using conditional statements
    Dynamic result output
  # Health Checker
    Basic symptom-based logic system
